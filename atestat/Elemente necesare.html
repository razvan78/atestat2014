  • - o placă Arduino UNO
  • - un cablu USB A-B (cablul este necesar doar pentru incărcarea codului )
  • - adaptor pentru baterie de 9V
  • - adaptor pentru Arduino la baterie
  • - ball-caster
  • - 2 roți
  • - cutie de viteze Tamiya (nu este necesară însă eu am folosit-o pentru a fixa mai bine roțile)
  • - 2 motoare DC de 6V
  • - 2 tranzistori ( BD139)
  • - un receptor infra-roșu
  • - o telecomanda ( orice telecomandă care emite semnal IR este buna )
  • - un regulator de voltaj de 6V ( acesta este opțional, eu l-am folosit ca masură de precauție pentru tranzistori)
  • - un breadboard
  • - fire izolate
  • - 4 clești
  • - piese LEGO (în cazul unor accidente caroseria este ușor de reparat)
